## 📝 题目描述

小码妹获得了远古神器 **阿斯特莱雅的天平**，她一共有 $n$ 颗魔法石，需要放置在天平两侧。

其中有 $m$ 条依赖关系：
- 互相依赖的魔法石**必须放在同一侧**
- 天平产生的魔力 = **左侧数量 × 右侧数量**

请你求出能获得的**最大魔力**。

---

## 🔽 输入格式

第一行输入两个整数 $n,m$。
接下来 $m$ 行，每行两个整数 $x_i,y_i$，表示 $x_i$ 与 $y_i$ 必须放在同一端。

**数据范围**
$1 \le n,m \le 2.5 \times 10^5$

---

## 🔼 输出格式

输出一个整数，表示最大魔力。

## 💡 题目分析

1. **依赖关系 = 连通块**
   使用 **并查集** 将必须同侧的魔法石合并成一个整体。

2. **连通块不可分割**
   每个连通块只能**整体放左边 或 整体放右边**。

3. **问题转化**
   从所有连通块中选择若干个，使总数量尽可能接近 $n/2$。

4. **数学结论**
   总和固定时，**两数越接近，乘积越大**。

---

## 🧠 方法一：标准 01 背包 DP

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 250010;
int fa[MAXN], sz[MAXN];
bool dp[MAXN];

int find(int x) {
    return fa[x] == x ? x : fa[x] = find(fa[x]);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    int n, m;
    cin >> n >> m;

    for (int i = 1; i <= n; ++i) {
        fa[i] = i;
        sz[i] = 1;
    }

    for (int i = 0; i < m; ++i) {
        int x, y;
        cin >> x >> y;
        x = find(x), y = find(y);
        if (x != y) {
            fa[y] = x;
            sz[x] += sz[y];
        }
    }

    dp[0] = true;
    for (int i = 1; i <= n; ++i) {
        if (find(i) == i) {
            int s = sz[i];
            for (int j = n / 2; j >= s; --j) {
                if (dp[j - s]) dp[j] = true;
            }
        }
    }

    int best = 0;
    for (int j = n / 2; j >= 0; --j) {
        if (dp[j]) {
            best = j;
            break;
        }
    }

    cout << 1LL * best * (n - best) << endl;
    return 0;
}
